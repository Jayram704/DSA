# DSA - Data Structures & Algorithms

A structured learning repository for mastering core DSA concepts and competitive programming. Solutions in **C++** with a focus on production-quality code and deep problem understanding.

## 📋 Structure

```
DSA/
├── arrays/              # Array manipulation, sliding window, two pointers
├── binary_search/       # Binary search patterns and edge cases
├── strings/            # String algorithms, pattern matching
├── linked_lists/       # Linked list operations and transformations
├── stacks_queues/      # Stack, queue, and deque problems
├── trees/              # Binary trees, BST, tree traversal
├── graphs/             # Graph traversal, shortest path, connectivity
├── dynamic_programming/ # DP optimization techniques
├── sorting/            # Sorting algorithms and comparators
├── hashing/            # Hash maps, hash sets, collision handling
└── misc/               # Math, bit manipulation, greedy problems
```

Each problem is standalone. To compile and run:

```bash

# Compile
g++ -o solution best_time_to_buy.cpp

# Run
./solution
```

Or use VS Code's C++ extension with MinGW for debugging.
